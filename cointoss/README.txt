REQUIREMENTS:
This program requires the GNU Compiler Collection (gcc) package.
http://gcc.gnu.org

RUNNING:
1. Extract the contents of this .tar.gz flie.
   (e.g., 'tar -xzvf cointoss_XXXXX.tar.gz')
2. Change to the 'cointoss_XXXXX' directory where the files were extracted.
3. Build and install the application by running the make utilities.
   (e.g., 'make && make install && make clean')

NOTE:
Once installed, usage information can be viewed by running 'cointoss --help'.

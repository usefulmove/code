REQUIREMENTS:
This program requires the GNU Compiler Collection (gcc) package.
http://gcc.gnu.org

RUNNING:
1. Extract the contents of this .tar.gz file.
   (e.g., 'tar -xzvf cnv_XXXXX.tar.gz')
2. Change to the 'cnv_XXXXX' directory where the files were extracted.
3. Build and install the application by running the make utilities.
   (e.g., 'make && make install && make clean' as root)

        make         -- compile project
        make install -- install application and man (manual) page
        make clean   -- remove program binaries following install

NOTE:
Once installed, usage information can be viewed by running 'cnv --help'.
